# Portfolio Advanced (PHP + MySQL on XAMPP)

## Setup
1) Copy this folder to: C:\xampp\htdocs\portfolio-advanced
2) Start Apache and MySQL in XAMPP.
3) Open http://localhost/phpmyadmin and run `php/create_tables.sql`:
   - Import the SQL file or paste the commands in SQL tab.
4) Visit: http://localhost/portfolio-advanced/login.html
   - Default credentials: admin / admin123 (change later).

## Replace with your assets
- Replace `/resume/resume.pdf` with your real resume (already pulled from your previous zip if detected).
- Replace `/images/profile.jpg` with your headshot.
- Replace `/images/project*.jpg` with real screenshots.

## Notes
- PHP is used only for DB connect and login.
- Front-end uses HTML/CSS/JS with a gradient hero, carousel, and responsive layout.
